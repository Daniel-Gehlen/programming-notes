    Tabela Completa de Fórmulas em Ordem Crescente de Dificuldade¶

    Tabela por ordem crescente de dificuldade lógica:

    Descrição Fórmula Matemática Notação Computacional
    Área de um retângulo A = b * h A = b * h
    Área de um triângulo A = (b * h) / 2 A = (b * h) / 2
    Volume de um cilindro V = πr²h V = pi * r^2 * h
    Seno de um ângulo sen(α) = cateto oposto / hipotenusa sen(alpha) = cateto_oposto / hipotenusa
    Cosseno de um ângulo cos(α) = cateto adjacente / hipotenusa cos(alpha) = cateto_adjacente / hipotenusa
    Tangente de um ângulo tan(α) = cateto oposto / cateto adjacente tan(alpha) = cateto_oposto / cateto_adjacente
    Cotangente de um ângulo cot(α) = 1 / tan(α) cot(alpha) = 1 / tan(alpha)
    Secante de um ângulo sec(α) = 1 / cos(α) sec(alpha) = 1 / cos(alpha)
    Cossecante de um ângulo csc(α) = 1 / sen(α) csc(alpha) = 1 / sen(alpha)
    Equação de primeiro grau ax + b = 0 a * x + b = 0
    Inequação de primeiro grau ax + b < 0 a * x + b < 0
    Função afim f(x) = ax + b f(x) = a * x + b
    Conversão de Celsius para Fahrenheit F = (9/5)C + 32 F = (9/5) * C + 32
    Equação de segundo grau ax² + bx + c = 0 a * x^2 + b * x + c = 0
    Inequação de segundo grau ax² + bx + c < 0 a * x^2 + b * x + c < 0
    Função quadrática f(x) = ax² + bx + c f(x) = a * x^2 + b * x + c
    Discriminante Δ = b² - 4ac Delta = b^2 - 4 * a * c
    Fórmula resolvente x = (-b ± √Δ) / 2a x = (-b ± sqrt(Delta)) / (2 * a)
    Distância entre dois pontos num plano d(P,Q) = √((x₂ - x₁)² + (y₂ - y₁)²) d_PQ = sqrt((x_2 - x_1)^2 + (y_2 - y_1)^2)
    Função seno f(x) = sen(x) f(x) = sin(x)
    Função cosseno f(x) = cos(x) f(x) = cos(x)
    Função tangente f(x) = tan(x) f(x) = tan(x)
    Equação exponencial aˣ = b a^x = b
    Inequação exponencial aˣ < b a^x < b
    Função exponencial f(x) = aˣ f(x) = a^x
    Equação logarítmica logₐ(x) = b log(x, a) = b
    Inequação logarítmica logₐ(x) < b log(x, a) < b
    Função logarítmica f(x) = logₐ(x) f(x) = log(x, a)
    Logaritmo da escala Richter M = log₁₀(A/A₀) M = log10(A / A_0)
    Variação no volume de um cilindro ∆V = 2πrh∆r + πr²∆h DeltaV = 2 * pi * r * h * Delta_r + pi * r^2 * Delta_h
    Número complexo na forma trigonométrica z(t) = 1000 _ cos(t) + 1000 _ sen(t) * i z(t) = 1000 * cos(t) + 1000 * sin(t) * i
    Módulo de um número complexo r = √(a² + b²) r = sqrt(a^2 + b^2)
    Derivada de uma função f'(x) = lim (f(x + h) - f(x)) / h (h→0) f'(x) = lim (f(x + h) - f(x)) / h (h -> 0)
    Integral indefinida ∫f(x)dx int(f(x), x)
    Integral definida ∫ₐᵇf(x)dx int(f(x), x, a, b)

    Observações:¶

    • A ordem das fórmulas é subjetiva e pode variar de acordo com a interpretação de cada pessoa sobre a dificuldade lógica.
    • As inequações podem ser escritas com outros operadores relacionais como >, ≤, ≥.
    • Esta tabela completa e organizada deve facilitar a consulta e o estudo das fórmulas.
