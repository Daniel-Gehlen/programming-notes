# Principais tipos de design de sites

### Uma explicação detalhada sobre os principais tipos de design de sites, desde os clássicos até os mais inovadores, com representações em ASCII art para visualizar cada layout.

## 1. DESIGN CLÁSSICO / TRADICIONAL

**Características:** Estrutura hierárquica clara, cabeçalho com menu, conteúdo principal, barra lateral.

**Indicado para:** Sites institucionais, blogs, portais de notícias.

```
+--------------------------------------------------+
|  LOGO                    MENU: Home | Sobre |   |
+--------------------------------------------------+
|  BANNER / DESTAQUE PRINCIPAL                     |
|                                                  |
+---------------+----------------------------------+
|               |                                  |
|   BARRA       |   CONTEÚDO PRINCIPAL             |
|   LATERAL     |   Artigo/Texto principal         |
|   -Categorias |   com imagens                     |
|   -Links      |                                  |
|   -Busca      |                                  |
|               |                                  |
+---------------+----------------------------------+
|  RODAPÉ - Contato | Redes Sociais | Endereço     |
+--------------------------------------------------+
```

## 2. DESIGN ONE PAGE / SCROLL

**Características:** Tudo em uma única página, navegação por âncoras, storytelling visual.

**Indicado para:** Portfólios, landing pages, eventos, produtos específicos.

```
+--------------------------------------------------+
|  LOGO                MENU (âncoras)              |
+--------------------------------------------------+
|                    HERO SECTION                   |
|              Título Principal                     |
|              Call-to-action                       |
|                                                  |
+--------------------------------------------------+
|                                                  |
|              SEÇÃO SOBRE                         |
|              Texto e imagem                       |

+--------------------------------------------------+
|                                                  |
|              SEÇÃO SERVIÇOS                       |
|    [Card]    [Card]    [Card]                    |
|                                                  |
+--------------------------------------------------+
|                                                  |
|              SEÇÃO PORTFÓLIO                      |
|    [Img]     [Img]     [Img]                     |
|                                                  |

|                                                  |
|              SEÇÃO CONTATO                        |
|    Formulário + Mapa                             |

+--------------------------------------------------+
|  RODAPÉ                                           |
+--------------------------------------------------+
```

## 3. DESIGN CARD-BASED / GRID

**Características:** Organização em cards, grid responsivo, muito usado em interfaces modernas.

**Indicado para:** E-commerces, portfólios, sites de conteúdo, dashboards.

```

+--------------------------------------------------+

|  | Título     |  | Título     |  | Título     |  |
|  | Descrição  |  | Descrição  |  | Descrição  |  |
|  | Preço      |  | Preço      |  | Preço      |  |
|  +------------+  +------------+  +------------+  |
|                                                  |
|  +------------+  +------------+  +------------+  |
|  | [IMAGEM]   |  | [IMAGEM]   |  | [IMAGEM]   |  |
```

## 4. DESIGN ASSIMÉTRICO / EDITORIAL

**Características:** Layout quebra a simetria, elementos sobrepostos, tipografia como destaque.

**Indicado para:** Revistas digitais, sites de moda, estúdios criativos.

```
+--------------------------------------------------+
|                    TÍTULO GRANDE                  |
|                EM DESTAQUE                        |
+------------------------+-------------------------+
|                        |                         |
|   [IMAGEM GRANDE]      |  TEXTO SOBREPOSTO      |
|                        |  COM ESTILO             |
|                        |                         |
+------------------------+-------------------------+
|     +----+                                        |
|     |    |  Texto flui ao redor                   |
|     +----+  da imagem em formato                   |
|            diferente                              |
|                                                   |
|  +----------+  +----------+                       |
|  | Citação  |  | Imagem   |                       |
|  | em       |  | pequena  |                       |
|  | destaque |  |          |                       |
|  +----------+  +----------+                       |
+--------------------------------------------------+
```

## 5. DESIGN SPLIT SCREEN

**Características:** Tela dividida em duas metades, cada uma com conteúdo diferente.

**Indicado para:** Sites comparativos, portfólios interativos, apresentações.

```
+--------------------------------+-----------------+
|                                |                 |
|                                |                 |
|          LADO ESQUERDO         |  LADO DIREITO   |
|                                |                 |
|         [IMAGEM/VÍDEO]         |  [CONTEÚDO]     |
|                                |  Título         |
|                                |  Texto          |
|                                |  Botão CTA      |
|                                |                 |
|                                |                 |
+--------------------------------+-----------------+
|           ELEMENTOS PODEM TROCAR                  |
|           CONFORME SCROLL/CLIQUE                   |
+--------------------------------------------------+
```

## 12. DESIGN DE CONTEÚDO IMERSIVO

**Características:** Conteúdo longo, elementos visuais entremeados, leitura fluida.

**Indicado para:** Blogs profissionais, medium-style, artigos longos.

```
+--------------------------------------------------+
|                                                  |
|                    TÍTULO                         |
|                                                  |
|              Subtítulo do artigo                  |
|                                                  |
|  [IMAGEM DE ABERTURA]                            |

|  naturalmente...                                 |

|  +----------------------------------+            |

|  |   destaque visual"                |            |
|  +----------------------------------+            |

+--------------------------------------------------+
```

## Tendências Atuais (2024-2025):

1. **Dark Mode** - Versões escuras como padrão
2. **Glassmorphism** - Efeito vidro fosco
3. **3D Elements** - Integração com WebGL
4. **AI-Personalization** - Conteúdo adaptativo
5. **Micro-animações** - Feedback visual constante
6. **Voice UI** - Interfaces de voz
7. **Sustainable Design** - Minimalista e eficiente

Cada tipo de negócio deve escolher o design que melhor comunica sua mensagem e atende às necessidades dos usuários, sempre considerando usabilidade, performance e objetivos de conversão.
