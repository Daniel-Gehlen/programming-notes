## As 4 fases da produção musical

1. Gravação
2. Edição
3. Mixagem
4. Masterização

O foco desta etapa é a **gravação**, com elementos já gravados separadamente (cada instrumento em sua própria faixa).

---

## Estratégia de gravação para home studio (exemplo do percurso)

| Elemento                        | Como será gravado                           |
| ------------------------------- | ------------------------------------------- |
| Bateria                         | Virtual (uso de instrumento virtual / MIDI) |
| Contrabaixo                     | Real                                        |
| Guitarra                        | Real, com simulador de amplificador         |
| Teclado / instrumentos virtuais | Virtual                                     |
| Violão                          | Real, com microfone                         |
| Voz                             | Real, com microfone                         |

---

## Como inserir uma nova faixa (track) no Reaper

No menu superior: `Track` > `Insert New Track`

Ou atalho: `Ctrl + T` (Windows) / `Cmd + T` (Mac)

---

## Elementos da interface de uma track (da esquerda para a direita, de cima para baixo)

| Elemento                           | Nome no Reaper          | Função                                                                                                       |
| ---------------------------------- | ----------------------- | ------------------------------------------------------------------------------------------------------------ |
| Botão vermelho                     | `Record Arm`            | Habilita a track para gravação. Sem ele ativado, nada será gravado.                                          |
| Medidor de volume (fader)          | `Track volume`          | Ajusta o volume de **audição** da track (não afeta o nível de gravação)                                      |
| Botão "Roteamento"                 | `Routing`               | Envia o sinal da track para outros lugares dentro do programa                                                |
| Botão "FX"                         | `FX`                    | Abre a janela para adicionar plugins (efeitos, simuladores, equalizadores, etc.)                             |
| Botão "Env" (Envelopes)            | `Envelopes`             | Cria parâmetros de automação (transições automáticas ao longo da música, ex.: volume subindo gradativamente) |
| Controle de posicionamento estéreo | `Pan`                   | Posiciona o som no eixo horizontal: esquerda (L), centro (C) ou direita (R)                                  |
| Botão "Mute" (M)                   | `Mute`                  | Silencia a track                                                                                             |
| Botão "Solo" (S)                   | `Solo`                  | Escuta **apenas** essa track (as outras ficam mudas)                                                         |
| Botão "Phase"                      | `Phase` (inverter fase) | Função mais avançada – será explicada posteriormente                                                         |

> Observação: o percurso trabalha com as configurações padrão do Reaper, sem customizações iniciais.

---

## Processo básico de configuração para gravação

1. **Criar uma nova track** (`Ctrl + T`)

2. **Armar a track para gravação** – clicar no botão vermelho `Record Arm`

3. **Selecionar a entrada correta** – clicar no seletor de entrada (ex.: `Input 1`), que corresponde ao canal da interface onde o microfone ou instrumento está conectado

4. **Monitorar o sinal** – falar ou tocar no microfone. O medidor de nível da track deve se mover

5. **Gravar** – clicar no botão `Record` na barra de transporte (círculo vermelho)

6. **Parar** – clicar em `Stop` (quadrado)

---

## Resumo visual

```
Track → Insert New Track (Ctrl + T)
├── Ativar Record Arm (botão vermelho)
├── Selecionar Input (ex: Input 1)
├── Verificar medidor de sinal
├── Clicar em Record (barra de transporte)
└── Clicar em Stop ao finalizar
```
