## Formato dos Arquivos

Cada arquivo markdown contém:
- **Título** da música
- **Artista** (quando identificado)
- **Tom** (quando disponível)
- **Cifra** completa com acordes
- **Letra** (quando disponível)
- **Fonte** da informação

## Origem dos Dados

Os arquivos foram convertidos automaticamente de documentos `.doc` e `.docx` presentes na pasta `assets/` do projeto, utilizando as ferramentas:
- `catdoc` para arquivos `.doc`
- `unzip` para arquivos `.docx`
